# Backend

Placeholder only. No backend exists yet.

Phase 10A (read-only UI display contract) deliberately adds **no** backend: no
server, no mutation endpoint, no action API. The read-only UI scaffold consumes
static `ui-view-model.v1` artifacts directly and needs no backend.

Any future backend would require a separate contract; see
[../docs/ui-readonly-contract.md](../docs/ui-readonly-contract.md).
