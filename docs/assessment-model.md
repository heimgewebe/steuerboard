# Assessment Model

Assessment derives status from observations, source refs, freshness, and rules.

An assessment must reference:

- the observation it used
- the source refs it used
- the freshness state of those sources
- the rule that produced the status
- the falsification case that motivated the rule, when applicable

## Boundary

Phase 0b defines only the minimum schema shape needed for future examples. Assessment execution begins after observation examples are grounded.
